# React Remotion Presets

A collection of reusable presets for creating videos with [React Remotion](https://www.remotion.dev/).

## Commands

**Install Dependencies**

```console
npm i
```

**Start Preview**

```console
npm run dev
```

**Render video**

```console
npx remotion render
```

**Upgrade Remotion**

```console
npx remotion upgrade
```
